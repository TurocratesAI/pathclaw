Welcome — your autonomous computational pathology lab.
Here is what you can ask me to do. I'll ask a couple of clarifying questions before starting big jobs.

🧬 Omics & genomics interpretation
• Parse MAF / VCF / clinical XML → mutation tables, TMB, MSI calls, per-gene queries
• Oncoplot of top mutated genes across a cohort
• Kaplan–Meier / log-rank survival stratified by mutation or model prediction
• Biomarker discovery (differential mutations between groups, attention-vs-mutation correlation)
• cBioPortal queries, STAR/HTSeq gene expression parsing, multi-omic label builder

🔬 WSI pipeline (end-to-end)
• GDC / TCGA cohort search & async download (slides + MAF + clinical + expression)
• Tissue segmentation & patching, feature extraction with UNI, Virchow2, CONCH, CTransPath, GigaPath
• MIL training — ABMIL, TransMIL, CLAM, DSMIL, RRTMIL, WIKG — with optional MAMMOTH MoE
• Segmentation training (U-Net, HoVer-Net, Cellpose); LoRA fine-tuning of foundation backbones
• Attention heatmaps, slide viewer with tile server; GPU-aware queue prevents race conditions

📚 Literature & writing
• Literature reviews with clickable PubMed / DOI / Semantic Scholar links — I'll first ask about scope, time window, paper type, depth
• Pull full-text PDFs into a session folder and summarise them
• Draft complete manuscripts in LaTeX (abstract → intro → methods → results → discussion → refs.bib)
• Embed plots straight from the Plots tab into the paper with one click (→ Manuscript)
• Compile to PDF in-session; export the raw .tex project or the compiled PDF

📦 Export
• Trained model .pth + config.json + metrics.json + plots as a single .zip
• Manuscript LaTeX project (sources + figures + compiled PDF) as .zip

Try: "Do a literature review on breast histopath subtyping — just SOTA from the last 3 years"
Or: "Extract MSI labels from tcga-ucec, run biomarker_discovery, generate an oncoplot of top 15 mutated genes, then draft a results section"

Connect Telegram

Message your agent from your phone. Each Telegram chat is bound to a PathClaw session — think of it as pinging your PhD students in different projects.

How to get a bot token (2 minutes):
  1. Open Telegram, search @BotFather, open chat
  2. Send /newbot
  3. Pick a name (e.g. "My PathClaw") and username (must end in _bot)
  4. BotFather replies with an HTTP API token like 7891234567:AAH... — copy it
  5. Paste it below and click Start bot
  6. On your phone, open your bot and send /sessions to see your running projects

Stored locally at ~/.pathclaw/config.json.

Comma-separated. Find your handle in Telegram → Settings → Username. Without this, anyone who finds your bot can talk to it.

If set, users must send /start <passcode> on first contact to authorize their chat.
